# 🏁 F1 Tracker 2026

A sleek, real-time Formula 1 season tracker built for the 2026 championship. All times displayed in **Philippine Time (PHT · UTC+8)**.

## Features
- **Next Race** — upcoming grand prix with full session schedule and live countdown
- **Championship Leader** — current standings leader with stats and driver photo
- **Last Race Podium** — top 3 finishers from the most recent grand prix
- **Live Standings** — full Drivers' and Constructors' Championship tables
- **Sprint Weekend detection** — automatically flags sprint format rounds
- **Mobile responsive** — works on desktop and phone

## Data
Powered by the [Jolpica API](https://jolpi.ca) (free, no API key required). Data updates 1–2 days after each race weekend.

## Built with
- Vanilla HTML / CSS / JavaScript
- Russo One + Barlow Condensed (Google Fonts)
- Jolpica F1 API
